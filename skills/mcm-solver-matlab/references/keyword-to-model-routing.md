# 关键词到模型路由

在正式建模前，先从题目中抽取关键词，再把关键词映射到候选模型。不要一上来直接写 MATLAB 代码。

## 使用规则

- 先提取题目中反复出现的目标词、约束词、对象词、评价词。
- 再给出 1 到 3 个候选模型，不要无限发散。
- 必须明确推荐模型，以及不优先采用其他模型的原因。
- 若题目包含多个子问题，可分别做关键词路由，再决定是否组合模型。

## 常见映射

### 优化类关键词

常见关键词：
- 最优
- 最大/最小
- 成本
- 收益
- 约束
- 调度
- 路径
- 分配
- 选址
- 方案设计

优先候选模型：
- 线性规划
- 整数规划
- 非线性规划
- 多目标优化
- 启发式算法

### 预测/拟合类关键词

常见关键词：
- 预测
- 趋势
- 未来
- 波动
- 拟合
- 回归
- 时间序列
- 补全
- 估计

优先候选模型：
- 线性/非线性回归
- 多项式拟合
- 时间序列模型
- 插值模型

### 统计分析类关键词

常见关键词：
- 相关性
- 显著性
- 影响因素
- 差异
- 分类
- 聚类
- 降维
- 识别

优先候选模型：
- 假设检验
- 相关与回归分析
- 聚类分析
- 主成分分析
- 分类模型

### 微分方程类关键词

常见关键词：
- 传播
- 演化
- 变化率
- 动态
- 稳态
- 初值
- 边界条件
- 机理

优先候选模型：
- 常微分方程
- 刚性微分方程
- 偏微分方程
- 动力系统模型

### 图论/网络类关键词

常见关键词：
- 节点
- 边
- 网络
- 连通
- 最短路径
- 流量
- 路网
- 拓扑

优先候选模型：
- 最短路径模型
- 最大流/最小割
- 最小生成树
- 网络中心性分析

### 评价模型类关键词

常见关键词：
- 评价
- 排序
- 指标体系
- 综合得分
- 权重
- 优劣
- 决策

优先候选模型：
- TOPSIS
- 熵权法
- 层次分析法
- 模糊综合评价

### 仿真类关键词

常见关键词：
- 模拟
- 仿真
- 重复试验
- 随机
- 场景
- 排队
- 概率
- 稳健性

优先候选模型：
- 蒙特卡罗仿真
- 离散事件仿真
- 基于规则的多轮仿真

## 组合判断

- 若同时出现“预测”和“最优”，通常先预测，再优化。
- 若同时出现“评价”和“决策”，通常先构建指标体系，再做排序或方案优选。
- 若同时出现“动态变化”和“随机扰动”，考虑把微分方程与仿真结合。
- 若同时出现“网络结构”和“资源调度”，考虑图论模型与优化模型结合。

## 输出要求

默认先输出以下内容，再进入正式建模：

```markdown
# 关键词与模型路由
- 关键词：...
- 候选模型：...
- 推荐模型：...
- 选择理由：...
- 不优先模型：...
```
