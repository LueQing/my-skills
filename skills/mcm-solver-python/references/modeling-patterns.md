# 建模模式速查

以下面向常见数学建模题型给出选型信号、Python 常用库、默认输出要素和典型陷阱，主技能在决策时可以直接引用此处内容。

## 1. 优化
- 适用场景：资源分配、调度、网络流、运筹等需要极小化成本或极大化收益的问题；目标函数、约束明确、可用解析表达式。
- Python 库：`scipy.optimize`（`minimize`、`linprog`）、`pulp`、`pyomo`、`ortools`、`deap`
- 默认输出要素：目标函数值、最优变量向量、约束满足情况、收敛状态、灵敏度或参数扰动结果、收敛历史图。
- 常见坑：未规范化量纲导致优化路径不稳定；整数约束和连续约束混用时未显式建模；梯度不可得时仍强行使用局部梯度法。

## 2. 预测 / 拟合
- 适用场景：时间序列预测、回归建模、机器学习回归任务，需要拟合输入与输出之间的关系。
- Python 库：`scikit-learn`、`statsmodels`、`scipy.interpolate`、`scipy.optimize.curve_fit`、`pandas`
- 默认输出要素：模型系数、`R^2` / `RMSE`、残差分布、预测区间、数据拆分方式（训练/验证/测试）、关键诊断图。
- 常见坑：未处理多重共线性、变量未标准化；忽略异方差或自相关；未固定随机种子导致结果不可复现。

## 3. 统计分析
- 适用场景：假设检验、分组对比、蒙特卡洛稳健性、敏感度分析。
- Python 库：`scipy.stats`、`statsmodels.stats`、`numpy.random`、`seaborn`、`matplotlib`
- 默认输出要素：检验统计量、p 值、置信区间、可视化（箱线图、直方图、QQ 图）、重抽样结果。
- 常见坑：忽略样本量小的偏差；多次比较时未校正；随机抽样未固定种子或未说明抽样策略。

## 4. 微分方程
- 适用场景：动力系统、传播模型、热传导、生态模型；需要求解初始/边值问题的连续变化。
- Python 库：`scipy.integrate.solve_ivp`、`scipy.integrate.odeint`、`scikits.odes`、`sympy`
- 默认输出要素：时间步列表、状态变量曲线、稳定性指标、误差控制参数、刚性问题下的求解器选择说明。
- 常见坑：未设置误差容忍度；初值或边界条件不合理导致发散；刚性问题仍使用不合适的显式方法。

## 5. 图论 / 网络
- 适用场景：路径优化、网络流、社交网络影响力分析、城市模型。
- Python 库：`networkx`、`igraph`、`numpy`、`matplotlib`
- 默认输出要素：图结构（节点/边）、度分布、关键路径、网络中心性、可视化图、网络健壮性分析指标。
- 常见坑：边权重与方向处理不一致；未检查图是否连通；大型图直接全量绘图导致性能问题。

## 6. 评价模型
- 适用场景：指标体系构建（熵权、层次分析、TOPSIS）、绩效评分、风险排序。
- Python 库：`numpy`、`pandas`、`scikit-learn`、`pymcdm`、`ahpy`
- 默认输出要素：权重矩阵、排序结果、稳定性敏感度、说明性表格、雷达图或条形图展示。
- 常见坑：权重没有归一化；指标方向未统一；未说明标准化方法（极差、Z-score、向量归一化）。

## 7. 仿真
- 适用场景：离散事件仿真、车间调度仿真、人群流动、系统动力学；强调时序演化和随机性。
- Python 库：`simpy`、`numpy`、`pandas`、`matplotlib`、`mesa`
- 默认输出要素：迭代步骤、随机种子、指标随时间变化、仿真快照、关键统计图或热力图。
- 常见坑：忽略仿真步长或事件调度对结果的影响；输出大量中间数据导致内存飙升；未保留可复现实验配置。
